# My Todo List API

A basic REST API to create, read, update and delete "To Do" items stored in a MongoDB database.

Made with [FeathersJS](https://feathersjs.com/).

Features:

- [x] CRUD operations
- [x] Search by title
- [x] Testing with Jest

## Commands

Start the server

```
npm start
```

Deploy on [Now.sh](https://zeit.co/now)

```
npm run deploy
```

Run the tests

```
npm test
```

Run the tests in watch mode

```
npm run test:watch
```
